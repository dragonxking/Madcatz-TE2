# Madcatz-TE2

## 内容：美加狮 TE2+ 街机摇杆面板图纸
## 用途：可用于定制铝合金面板代替原来的亚克力透明面板
## 经验：
1. 加工前请 1:1 打印出来与原来的亚克力面板进行实物对比，以便确认图纸尺寸是否合适，再进行加工
2. 使用 2mm 厚的铝合金面板，进行氧化喷砂加工。
3. 由于铝合金材质较软，且厚度仅有 2mm，加工厂家会提示有不平整的风险，但实际使用可以忽略
